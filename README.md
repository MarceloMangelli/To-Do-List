# 📝 To-Do List

A simple task management application built as a learning project to practice JavaScript and DOM manipulation.

---

## 🚀 About the Project

This project was developed as a practical exercise to strengthen core front-end development skills, especially JavaScript logic and DOM manipulation.

The application allows users to organize tasks in a simple way by adding items to a list and tracking their completion status.

---

## ⚙️ Features

- ✅ Add new tasks  
- ✅ Mark tasks as completed  
- ✅ Delete individual tasks  
- ✅ Clear all tasks  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)

---

## 💾 Data Persistence

Currently, the application **does not store data**.  
All tasks are lost when the page is refreshed.

---

## 🎨 Interface

The interface is simple and functional, focusing on usability and clarity of user actions.
